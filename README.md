## Installation Steps:
1. Use [bun](https://bun.sh) or any other package manager to install all the dependencies:
```
npm install 
```

2. Build:
```
npm run build
```

3. Rename [.env.example](https://github.com/jpcrs/tvbarrapesada/blob/main/.env.example) to .env

## Usage
Start:
```
npm run start
```
